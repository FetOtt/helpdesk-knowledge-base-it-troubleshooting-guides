# Basic Network Troubleshooting

## Quick Layer Checklist
- Layer 1: Cables, link lights
- Layer 2: Switch/Wi-Fi AP
- Layer 3: IP, Gateway, DNS

## Commands
- `ipconfig /all`
- `ping 8.8.8.8`
- `ping google.com`
- `tracert google.com`

## Common Issues
- Wrong gateway
- DNS failure
- DHCP not giving IP
- Blocked ports
