# Common Port Numbers Cheat Sheet

| Port | Protocol | Description |
|------|----------|-------------|
| 20/21 | FTP | File Transfer |
| 22 | SSH | Secure Shell |
| 23 | Telnet | Remote login (unsecure) |
| 25 | SMTP | Outgoing email |
| 53 | DNS | Name resolution |
| 67/68 | DHCP | Auto IP assignment |
| 69 | TFTP | Simple file transfer |
| 80 | HTTP | Web browsing |
| 110 | POP3 | Email retrieval |
| 123 | NTP | Time sync |
| 143 | IMAP | Modern email retrieval |
| 161/162 | SNMP | Network monitoring |
| 389 | LDAP | Directory services |
| 443 | HTTPS | Secure web |
| 445 | SMB | File sharing |
| 3389 | RDP | Remote Desktop |
