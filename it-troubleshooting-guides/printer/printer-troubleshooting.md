# Printer Troubleshooting Guide

## 1. Identify the Issue
- Local or network printer?
- Error messages?

## 2. Hardware Checks
- Power
- Paper
- Toner/ink
- Connection cables

## 3. OS-Level Checks
- Default printer set?
- Clear print queue
- Restart Print Spooler

## 4. Network Checks
- Ping printer IP
- Is printer online?
- Re-map shared printer

## 5. Driver Issues
- Update or reinstall drivers
- Check user permissions

## 6. Document
- Root cause
- Fix applied
