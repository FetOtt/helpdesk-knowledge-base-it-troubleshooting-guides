# Hardware Triage Checklist

## Power Checks
- Plugged in?
- Power brick working?

## Internal Checks
- Reseat RAM
- Check fans
- Inspect motherboard cables

## Boot Issues
- BIOS/UEFI displays?
- POST beeps?

## Storage
- HDD/SSD health
- Loose SATA cables

## Document
- Symptoms
- Fix
- Recommendation
